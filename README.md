# 🕹️ Tic-Tac-Toe Game 
A classic implementation of the Tic-Tac-Toe game, developed as part of my programming projects at **Fatima Jinnah Women University**.

---

### 📝 Project Description
This is a functional, command-line (ya GUI, agar aapne GUI banaya hai toh likhein) based game where two players can play against each other. It focuses on core programming logic, array manipulation, and win-condition algorithms.

### 🚀 Features
- **Two-Player Mode:** Play with a friend on the same system.
- **Input Validation:** Prevents players from choosing an already occupied spot.
- **Win Detection:** Automatically detects Horizontal, Vertical, and Diagonal wins.
- **Draw Condition:** Detects when the board is full and no one has won.

### 🛠️ Tech Stack
- **Language:** C++ / Python (Jo aapne use ki hai, sirf wo rehne dein)
- **Concepts used:** Arrays/Lists, Functions, Loops, and Conditional Logic.
